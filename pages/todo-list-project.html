1. Introduction

1.1 Contexte et Objectifs

Le projet "TODO List" est une application de gestion de tâches avec une interface utilisateur moderne. L'objectif principal était de développer une solution complète en utilisant JavaFX pour l'interface graphique, tout en implémentant un système de stockage local pour les données.

Cette application répond à un besoin fondamental d'organisation et d'optimisation des tâches quotidiennes, permettant aux utilisateurs de gérer efficacement leurs activités.

1.2 Technologies Utilisées

Java JavaFX Fichiers Texte/DAT PlantUML/UML IntelliJ

2. Architecture de l'application

2.1 Structure Générale

L'application est organisée en 4 packages principaux :

  • com.todo.model : Modèles (User, Task)
  • com.todo.persistence : Gestion des fichiers/bases de données (Database, TaskStorage)
  • com.todo.controller : Gestion des actions utilisateur (TaskManager)
  • com.todo.app : Interface utilisateur (AdminPanel, MainApp, LoggingPage)

2.2 Diagramme UML de l'Architecture

L'architecture a été modélisée à l'aide de PlantUML, illustrant clairement les relations entre les différents composants du système.

3. Fonctionnalités Implémentées

3.1 Affichage des Tâches

Utilisation de TableView et ListView pour afficher les tâches avec leurs propriétés (titre, date limite, durée, état de complétion).

3.2 Ajout de Tâches

Interface intuitive permettant d'ajouter de nouvelles tâches avec leurs caractéristiques : nom, durée, date limite et catégorie.

3.3 Marquage d'une Tâche comme Complétée

Système de CheckBox permettant de modifier facilement l'état de complétion d'une tâche.

3.4 Suppression des Tâches

Bouton de suppression (❌) intégré pour chaque tâche, permettant une gestion efficace de la liste.

3.5 Filtrage des Tâches

Système de filtrage via ComboBox offrant trois options : "Toutes", "Complétées", "Non complétées".

3.6 Interface Administrateur

Panel d'administration (AdminPanel) permettant la gestion des utilisateurs, notamment la possibilité de supprimer des comptes utilisateur depuis l'interface.
